# Cooking-timer
This is an app that alerts you once your food is done cooking
## Application 
This app is applicable to any type of food, including fried foods.
## Features 
- Adjustable time 
- Food type Selection 
- Food done Alarm 
## Advantage 
1. you can do something else without the fear of your food getting burnt
